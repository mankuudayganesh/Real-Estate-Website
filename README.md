# Aurelia Luxury Real Estate Website

An ultra-premium, agency-grade luxury real estate frontend website built using pure **HTML5**, **CSS3**, and **Vanilla ES6 JavaScript**.

## ✨ Design Style & Aesthetic
* **Luxury UI**: Spacious layouts, high contrast dark/light combinations, and fine lines.
* **Glassmorphism**: Glass-morphed panels (`backdrop-filter`) for headers, cards, and interactive tabs.
* **Premium Typography**: Classical headings via *Playfair Display*, geometric body paragraphs via *Poppins*, and clean uppercase actions via *Inter*.
* **Harmonious Palette**: 
  - Slate Dark Primary: `#0F172A`
  - Slate Medium Secondary: `#1E293B`
  - Metallic Gold Accent: `#D4AF37`
  - Light Gray Backgrounds: `#F8FAFC`
  - White: `#FFFFFF`
  - Text Gray: `#64748B`

---

## 📂 Page Structure (6 Total Pages)
1. **Home (`index.html`)**: Parallax villa hero section with active stats, 6 property cards with lift animations, testimonial sliders, video showcase, and custom footer.
2. **Properties Portfolio (`properties.html`)**: 12-listing grid, sticky query filters (keyword search, location select, property type, budget, and bedroom numbers), and horizontal investment banners with ROI badges.
3. **Property Details (`property-details.html`)**: Clickable image swapper gallery, specs grid, interactive tabs (Overview, Amenities checklist, Location overview, Floor Plan drawing), and a dedicated Agent sidebar profile.
4. **About Us (`about.html`)**: Historical growth timeline, Mission & Vision glass cards, executive team cards, and count-up achievement statistics.
5. **Our Services (`services.html`)**: Buying/Selling/Management service items with gold hover glows, 5-step methodology chart, and collapsible FAQ Accordions.
6. **Contact Us (`contact.html`)**: Contact details, floating label form with live validation checks, office locations, and an abstract global coordinates map layout with pulsing gold pings.
